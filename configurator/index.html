This application uses a JSON file to keep track of your classes.
You can create a schedule specification in your favorite text editor
following the instructions below. 

[ // This is the array containing all periods.
	{ // This is one period.
		// "name" will be shown on-screen.
		"name": "PE",
		// If "url" is present, "name" will be a link to "url".
		"url": "https://meet.google.com/...",
		// "regex" is a regular expression that matches ETHSBell's name for this period.
		"regex": "^\\D*2\\D*$", // Matches 2nd period.
		// "pe" is the last name of the PE teacher that teaches this class, if any.
		// It is used to integrate with the PE Board.
		"pe": "Brewer"
	}
]
